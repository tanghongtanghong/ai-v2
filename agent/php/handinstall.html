PHP Agent 手动安装

  • Linux 手动安装
  • windows 手动安装

Linux 手动安装

一、获取相关信息

1.获取 php 版本信息

php -v

2.获取 php 扩展目录

php -i|grep -i "extension_dir“

3.查看 php 是否是线程安全版本

php -i|grep -i "Thread Safety”

4.查看系统为 x86 还是 x86_64

从上面的操作可以知道

PHP 版本 ： 5.5.15

PHP 扩展目录：

/usr/local/php5/lib/php/extensions/no-debug-non-zts-20121212

PHP 是否为线程安全版本 ： 否

系统：x86_64

二、安装

PHP 探针分为两部分：oneapm-daemon 和 PHP 扩展两部分（oneapm.so）

PHP 扩展部分负责搜集数据，oneapm-daemon 负责合并数据并定时发送给 Server 端。

下载安装包到本地后，解压，并进入安装目录

tar zxvf OneAPM_PHP_Agent_2.5.1-b.tar.gz && cd oneapm-php-install-b

查看目录结构

  • 1. 安装 PHP 扩展 ( oneapm.so)

根据第一步得到的信息，我们需要的 PHP 扩展为 agent/x64 下的 oneapm-php5.5.so

\cp -p agent/x64/oneapm-php5.5.so 

/usr/local/php5/lib/php/extensions/no-debug-non-zts-20121212/oneapm.so

修改 php.ini 文件，添加

oneapm.key="Your License"
extension=oneapm.so
oneapm.appname="PHP Application"
oneapm.tier_name="PHP_TIER" （企业版专有）
oneapm.cross_application_tracer.enabled=true （企业版专有）

创建 log 目录

mkdir -p /var/log/oneapm/ && chmod 777 /var/log/oneapm/
  • 2. 安装 oneapm-daemon

创建 oneapm-daemon 配置文件目录

mkdir -p /etc/oneapm/ && chmod 755 /etc/oneapm

拷贝 oneapm-daemon 配置文件

\cp -p scripts/oneapm.cfg.template /etc/oneapm/oneapm.cfg

拷贝 oneapm-daemon 二进制程序

\cp -p daemon/oneapm-daemon-x64 /usr/bin/oneapm-daemon

修改 vi /etc/oneapm/oneapm.cfg 中的配置 （企业版）

collector_host=Your Enterprise Server IP
port=Your port

三、 启动

  • 1. 启动，停止，查看 oneapm-daemon

启动：直接在命令行输入 oneapm-daemon 回车

停止：oneapm-daemon -s stop

查看：ps -ef|grep oneapm-daemon

  • 2. 重启 Apache 或 php-fpm 使 oneapm.so 加载到 PHP 中

service httpd restart 或

service php-fpm restart

windows 手动安装

一、获取 php 及系统信息

  1. 在开始菜单输入 cmd

看到 cmd.exe,右键 cmd.exe，以管理员身份运行，弹出命令窗口。

2. 获取 php 版本以及是否线程安全

如果 php 安装路径没有加入环境变量，则先进入php的安装目录(在命令行执行 “cd php 安装路径” ),如果已经加入环境变量，则直接执行一下命令

  • 版本信息：在命令行执行 php -v

从上面操作可知 php 的版本是 5.6.25

  • 是否线程安全：在命令行执行 php -i |find /i “thread”

从上面的操作可知，php 是线程安全的\

二、下载安装探针

1.下载探针 zip 包

在官网下载手动安装 zip 包：OneAPM_php_Agent_xxx.zip

2.解压，进入 windows-installer 目录

3.获取 php 位数

进入安装包的 dumpbin 目录，看到 dump.exe 文件

将 dump.exe 拖入刚刚打开的cmd窗口，空格

然后进入 php 安装目录，将 php.exe 也拖入命令行窗口

回车

由上图可知是 php 是 64 位

4.安装扩展

进入 windows-installer\agent 目录

根据自己的 php 位数进入目录，如进入 x64

根据自己的php版本将合适的dll文件复制到php安装目录下的 ext 文件夹中

5.修改 php.ini

进入 php 安装目录，打开 php.ini 文件

修改 php.ini 文件，添加

[oneapm] oneapm.key=“Your License”
extension=D:\php-win\iis55\php-5.5.21-nts-Win32-VC11-x64\ext\oneapm-php5.5.dll （上一步中拷贝的dll文件）
oneapm.appname=“PHP Application”
；oneapm.tier_name=“PHP_TIER” （企业版专有）
；oneapm.cross_application_tracer.enabled=true （企业版专有）
oneapm.loglevel=“info”

6.安装 daemon

进入windows-installer\daemon目录

根据自己的操作系统位数进入目录，如进入x64

将这个文件夹拷入C:\ProgramData目录中

7.注册 oneapm-daemon 服务

进入

C:\ProgramData\oneapm-daemon

目录修改oneapm-daemon-x86.exe名称为oneapm-daemon.exe

在开始菜单输入cmd，右键cmd，以管理员身份运行，输入以下命令：

sc create "OneAPM PHP Daemon" binPath= "C:\ProgramData\oneapm-daemon\oneapm-daemon.exe"

如上出现 CreateService 成功的时候，证明注册服务成功

8.重启 Apache 或 php-fpm 使 oneapm.so 加载到 PHP 中，启动 daemon

开始菜单输入 services.msc 回车

  • 重启服务器或 fastcig

如果使用 apache 作为服务器，重启 apache，在弹出的服务对话框中找到 apache

右键，重新启动

如果使用 nginx 作为服务器，重启你所使用的 fastcgi

  • 启动 oneapm php daemon 服务，在上一步弹出的对话框中找到 OneAPM PHP Daemon

右键，重启
